# THE COMPLETE USER GUIDE
## MASTERING METADATA WITH EDXIF PRO

### 1 WELCOME TO THE TERMINAL
This guide provides an exhaustive walkthrough of every feature available in EDXIF PRO. Whether you are a professional photographer a forensic analyst or a technical hobbyist this manual will help you navigate the complexities of digital metadata.

### 2 THE NAVIGATOR PANE
The left side of the screen is your primary command center for file management.
1. IMPORTING ASSETS
Click the large upload area to choose a file or simply drag and drop your file into the window.
2. CAMERA CAPTURE
If your device has an integrated camera you can use the capture button to take a fresh photo directement into the buffer.
3. FILE DESCRIPTORS
After a file is loaded the descriptor section provides quick stats including the filename and total size in megabytes.

### 3 THE METADATA EDITOR
The central area is where the analysis and editing occur.
1. DATA CATEGORIES
Use the tabs at the top to filter tags. The camera tab shows photography settings while the GPS tab shows location data.
2. EDITING VALUES
The value field for each tag is clickable. When you enter a new value it will highlighted to indicate a pending change.
3. COMMITTING CHANGES
Metadata changes stay in the temporary buffer until you click the COMMIT button. This triggers the engine to write the new data into the file.

### 4 VISUAL CUSTOMIZATION
EDXIF PRO offers a diverse range of visual environments.
1. PALETTE SELECTION
Access the palette menu from the top bar to choose from over 30 different themes including classic terminal looks and modern minimal designs.
2. RESPONSIVE LAYOUT
The interface automatically adjusts for mobile devices. On smaller screens use the menu icon to toggle the navigator pane.

### 5 TROUBLESHOOTING
1. FILE NOT RECOGNIZED
If the system fails to extract metadata ensure that your file is not corrupted and is in a supported format like JPG or MOV.
2. CHANGES NOT SAVING
Ensure you have clicked the COMMIT button before trying to EXPORT. The export function only saves the last committed state.
3. GEOLOCATION FAILS
If you are trying to use your current location ensure that you have granted the browser permission to access your GPS data.

### 6 ADVANCED TIPS
For the best results always audit the system tags first to understand how the file is structured before applying modifications. Forensic precision requires a thorough understanding of the original data state.
